# MiniproyectoPHP
Proyecto hecho en PHP, HTML y MySql.
Este miniproyecto tenia como funcion crear, consultar, borrar y actualizar, una lista de estudiantes y profesores, ademas de eso tener como referencia facultades y programas de una universidad.
Saber a que facultad pertenencen, a que programa estan inscritos, con que profesores y cual es el estudiante.
